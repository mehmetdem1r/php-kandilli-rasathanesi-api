# php-kandilli-rasathanesi-api
PHP Kandilli rasathanesi API

Gelen verileri db üzerinde tutabilir ve kapsamlı raporlama sunabilirsiniz.
- 143,741 adet deprem verisi, veritabanı olarak eklenmiştir.
